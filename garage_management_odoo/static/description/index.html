Garage Management System with Job Card

Garage Management Odoo App with Job Card / Estimation / Requisition / Vehicle Details / Timesheets / Checklist / Instructions

This app allows the garage manager (project manager) and project users (garage user) to manage the garage using a job card with vehicle details, cost sheet, timesheet, checklist, instruction, invoicing etc as shown in below screenshots.

Main Features:

  • Garage (Project) managers / Project users / Technicians can fill the job card with vehicle details.
  • Customers can accept a cost sheet estimate for vehicle repairing from the portal of the website.
  • The manager can give instructions to the technician and users.
  • Users can see instructions given to them and also tick the checklist as per work done.
  • You can create an invoice for vehicle servicing/repairing as shown.
  • Managers can set the type of vehicle for which the garage provides service.
  • Manager can also set the repair category for the garage.
  • For more details please check the video and check the below screenshots.
